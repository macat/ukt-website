# UKT Air Conditioners — Modern Website

A modern, professional website for **United Klima Technologies (UKT)**, a Michigan-based manufacturer of HVAC systems that require no outdoor condenser unit.

## Features
- Clean, dark industrial design with modern typography
- Animated SVG airflow diagrams showing how the units work
- Full technical specifications from official data sheets
- Responsive design for mobile and desktop
- Product showcase for all UKT models

## Products Covered
- **Attic Compact** — Ceiling-mount ductable HVAC (10" profile)
- **Vertical Ductable** — Wall-strip unit with hydronic option
- **Vertical HVAC** — Full-height cabinet unit
- **Baby HVAC** — Ultra-compact for tight spaces

## Tech Stack
- Pure HTML/CSS/JS (no frameworks)
- Google Fonts (Inter, JetBrains Mono)
- Inline SVG animations
- CSS animations for airflow visualization

## Deploy
Hosted on GitHub Pages.
